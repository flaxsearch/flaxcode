Description
The description will be visible to end users.
Files Specification

The options in this section determine which files will be included in this document collection.

Paths
The directories specified here will be (recursively) searched for files. Individual files may also be specified here.
Formats
Files of the selected formats will be included.
Exclusions
Patterns matching exclusions will be ommitted from the files specified in paths.
Earliest
Files modified before this date will be excluded.
Latest
Files modified after this date will be excluded.
Oldest
Files older than this duration at the time of indexing will be excluded
Youngest
Files newer than this duration at the time of indexing will be excluded.
Indexing Options
These options control the way this document collection is indexed.
Scheduling
These options determine when indexing of this collection takes place.
