Logging

These options controls which events are logged.

Events have both a category and a severity. For each category you can specify the level at or above which events are logged. For example, the "Critical" level will cause only critical errors to be logged, whereas the "Debug" level will cause all messages including debugging message to be logged (which may result in very large number of log entries being generated). The "Default" level will cause the level for a particular category of log message to follow the setting for the "default" category.
