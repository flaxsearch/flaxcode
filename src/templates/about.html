Flax Basic

Version .

Flax Basic is a simple enterprise search system, designed to allow you to build searchable indexes of files on your PC, intranet or website. Flax Basic is based on Lemur Consulting's search solution platform, Flax. Further support is available at the Flax website.

Flax is open source software, with no license fees, meaning that its source code is freely available. Flax is based on Xapian, a high-performance, cross-platform and extremely accurate technology used across the world.

The Flax search solution platform allows for the development of advanced search systems with features that include:

  • Windows, Mac and Linux platforms
  • Build searchable indexes of:
    • Microsoft Office (Word, Excel, Powerpoint), Adobe Acrobat, HTML, RTF and plain text documents, and many other common formats
    • Emails and content stored in Microsoft Exchange and other email systems
    • Webpages from your own or external websites via controllable spiders
    • Records extracted from existing databases (Microsoft Access, SQL, Oracle etc.)
  • Give your users powerful search facilities including:
    • Boolean search operators (AND, OR, NOT)
    • Probabilistic Bayesian, wildcard, phrase and proximity search
    • Faceted search and support for user tags
    • Automatic spelling correction of user queries
    • Thesaurus and synonym support
    • Search while an index is being updated - new documents are instantly searchable
    • Auto-suggest extra search terms & related documents
    • Restrict search results by custom field, date or numeric range
    • Sort search results by relevance, date or custom fields
    • Simple or Advanced search pages using a fully customisable interface
    • Stemming for 14 different European different languages
  • Administrators have full control of the system using
    • Web-based administration system with secure logon
    • View statistics and logs
    • Create multiple collections of indexed data
    • Scheduled or manually controlled indexing
    • Map source files to URLs
    • Create custom metadata fields
    • Fully customisable templates to for search pages and results
    • Export results as XML
    • Simple integration into existing architectures
